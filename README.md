# 추첨권 계산기

감점 방식으로 평가된 점수를 기반으로 추첨권을 분배하고, 가중 확률 돌림판으로 당첨자를 선정하는 도구.

**[→ 바로 사용하기](https://lethalsun.github.io/lottery-calc/)**

---

## 사용 사례

선거 공약, 업무 평가 등 **감점 방식으로 점수를 매긴 후보들 중 랜덤하게 하나를 선택**할 때 사용합니다.  
가점 방식이 무의미한 분야에서 감점 점수를 확률로 변환하여 공정한 추첨을 진행합니다.

---

## 핵심 알고리즘: Softmax

점수를 추첨 확률로 변환할 때 **Softmax** 함수를 사용합니다.

$$P_i = \frac{e^{s_i \cdot T}}{\sum_j e^{s_j \cdot T}}$$

- $s_i$: 후보 $i$의 점수 (음수)
- $T$: Temperature 파라미터

점수가 높을수록(덜 깎일수록) 더 높은 확률을 가집니다.

### Temperature(T) 역할

| T 값 | 효과 |
|------|------|
| 높음 | 점수 차이가 강하게 반영됨 (상위 후보 독식) |
| 낮음 | 점수 차이 무시, 균등에 가까워짐 |

**권장 T 자동 계산:**
$$T = \frac{1}{\text{최고점} - \text{최저점}}$$

점수 범위에 따라 적절한 차등이 유지되도록 자동 조정됩니다.

## 검증

**시뮬레이션 후 추첨** 버튼을 누르면 10,000회 시뮬레이션을 실행하여 이론 확률과 실제 분포를 비교합니다.

| 후보 | 기댓값 | 실측 |
|------|--------|------|
| A(-55) | 97.x% | 97.x% |
| B(-420) | 2.x% | 2.x% |

오차 1.5%p 이내이면 정상 동작입니다.

---

## 사용법

1. 점수 입력란에 음수 점수를 쉼표로 구분해 입력 (예: `-55, -180, -420`)
2. 후보 이름 입력 (선택)
3. **계산** 버튼 클릭
4. T 슬라이더로 차등 강도 조절 또는 **권장 T 자동 설정**
5. **돌리기** 또는 **시뮬레이션 후 추첨**
